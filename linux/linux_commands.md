# Linux Commands for Application Support

## pwd

Displays the current working directory.

Example:

pwd

---

## ls

Lists files and directories.

Example:

ls

ls -l

ls -la

---

## cd

Changes directory.

Example:

cd /var/log

cd ..

---

## mkdir

Creates a new directory.

Example:

mkdir logs

---

## cp

Copies files.

Example:

cp source.txt destination.txt

---

## cat

Displays file contents.

Example:

cat application.log

---

## head

Displays first lines of a file.

Example:

head application.log

head -20 application.log

---

## tail

Displays last lines of a file.

Example:

tail application.log

tail -f application.log

---

## grep

Searches text in files.

Example:

grep "ERROR" application.log

grep "Exception" application.log

---

## find

Searches for files and folders.

Example:

find /home -name "*.log"

---

## ps

Shows running processes.

Example:

ps -ef

---

## top

Displays real-time system usage.

Example:

top

---

## kill

Terminates a process.

Example:

kill 1234

kill -9 1234

---

## df

Displays disk space usage.

Example:

df -h

---

## chmod

Changes file permissions.

Example:

chmod 755 script.sh

---

## ping

Checks network connectivity.

Example:

ping google.com
